# Changelog

All notable changes to the OmniGuard CLI will be documented in this file.

## [1.0.0] - 2026-07-09

### Added
- Complete rewrite of CLI command handlers.
- Secure, transparent profile configuration encryption using AES-256-CBC.
- Native, dependency-free interactive TUI using ASCII graphics and ANSI sequences.
- Local scanning engines for Secrets, IaC configurations, Container Dockerfiles, and package dependency vulnerability audits.
- Enterprise integrations command hooks for Jira, ServiceNow, Confluence, Okta, and HashiCorp Vault.
- Fully automated publishing scripts for Windows Powershell and bash.
